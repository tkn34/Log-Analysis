# Log-Analysis
Log Analysis for AI

* **`実行方法`**
  1. jupyterNotebookを起動する。
  2. train.ipynbを開く。

* **`実行内容`**
  1. データ取得(公開データセット「BGL」を取得する。)
  2. データ前処理(正規表現を用いて日時やIPアドレスのようなログごとの固有値を削除)
  3. 特徴量抽出(tf-ilfを利用して特徴量抽出)
  4. LightGBMを用いて学習実行
  5. 学習時の精度(混同行列)を表示
  6. 特徴量重要度より、重要とする単語を抽出
  7. 推論の実行
  8. 推論時の精度(混同行列)を表示
  9. ログの可視化(異常と判定したログを赤で表示)
